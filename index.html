Intelligent Text Clustering

While issuing the rulings regarding some social or religious affair, the Islamic Jurists (legal experts) has to go into all the basic but authentic texts to find the stuff which is directly related or having the contextual analogy or has an association with it. It is obvious that the verdict or ruling of a jury or jurist will be strong if he takes into account all the contextually related stuff which can only be made possible by developing such a system which upon query infers a context, and produces before the jury all the related corpus. If the decision is taken just at the shallower level, it potentially leads to the rift and conflict in the society and has been historically observed. Deep learning techniques preferably Deep Embedded Clustering (DEC), BIRCH, Spectral were used to deploy models for contextual similarity and analogy. Embedding of the corpus were calculated using Google’s Word2Vec, Doc2vec, fastText and Stanford’s GloVe. The embeddings were calculated in itself from the text corpus using cosine similarity in the multidimensional context. Google’s Tensorflow along with Keras framework have been deployed to carry out the training of the models.

In this case study, we examined the effect of word embeddings on many existing clustering techniques and proposed and compared the new 16 embedding based clustering models. We implement the 16 different embedding based models.

Among all the proposed models, we conclude that Doc2Vec based clustering techniques provide a very good and dense clustering of the data based on our results. The Doc2Vec based clustering model outperforms the other embedding based clustering models. This is due to the reason that Doc2Vec based clus- tering takes context (neighbouring context) into consideration and based on the contextual similarity of the Hadith document, it performs better clustering than the other proposed ones and thus groups the relevant Hadith documents into the same cluster. We have also implemented the basic graphical user in- terface (GUI) for demonstration purposes only in which a Hadith is given as an input to the program and the program retrieves the relevant Hadiths which fall within the same cluster based on its contextual similarity. The applica- tion of which is in-terms of decision support system while issuing the rulings regarding some religious or social affairs in the Islamic Jurisprudence. Based on our results, we also conclude that the fastText based clustering shows the worst comparative results among the proposed models. MS Thesis Eng. MS Thesis TR. PPT. Code. Code.

NachOS

In addition, our system implements user semaphores and also some mechanisms to avoid common errors when writing multithread programmes, like forgetting to use a join before the main thread exits (and killing all the threads even if they have not finished) and doing a join to a thread that is not a child of the calling thread.
